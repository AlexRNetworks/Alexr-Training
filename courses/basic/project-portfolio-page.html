Final Project: Build Your Personal Portfolio Page

Congratulations! You've learned about HTML structure, CSS styling, and the box model. Now it's time to put it all together and build something real. Your task is to create a simple, elegant, single-page personal portfolio.

Project Requirements

Your finished page must include:

  • A main heading (<h1>) with your name.
  • A sub-heading (e.g., <h2> or <p>) with a title like "Aspiring Web Developer".
  • An "About Me" section with a heading and at least one paragraph.
  • A "Skills" section with a heading and an unordered list (<ul>) of skills you're learning.
  • A simple footer with your email or a copyright notice.
  • Custom CSS styling for colors, fonts, and spacing (using margin and padding).

Step 1: The HTML Structure

First, create the skeleton of your page. Use the semantic tags we learned about (<header>, <main>, <footer>, <section>) to organize your content logically. 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Your Name - Portfolio</title>
    <link rel="stylesheet" href="your-portfolio-style.css"> <!-- Important! -->
</head>
<body>

    <header>
        <!-- Your name and title go here -->
    </header>

    <main>
        <section id="about">
            <!-- Your "About Me" content goes here -->
        </section>

        <section id="skills">
            <!-- Your "Skills" list goes here -->
        </section>
    </main>

    <footer>
        <!-- Your footer content goes here -->
    </footer>

</body>
</html>

Step 2: The CSS Styling

This is where you get to be creative! Create a new file called your-portfolio-style.css. Don't forget to link to it from your HTML file as shown above.

Try to style your page. Experiment with the properties you've learned. Here are some ideas and hints:

  • Give the body a background color and a default font family.
  • Center the content by setting a max-width and using margin: 0 auto; on a main container div.
  • Style your sections by giving them `padding` and a `margin-bottom` to create space.
  • Choose a color palette for your headings and text.

Example CSS Snippet (A Hint!)

Here's a small example of how you could style one of your sections to get you started.

/* In your-portfolio-style.css */

body {
    font-family: 'Helvetica', sans-serif;
    background-color: #f4f4f4;
    color: #333;
}

#about {
    background-color: #fff;
    padding: 25px;
    margin: 20px 0;
    border-radius: 8px;
    border-left: 5px solid #0056b3; /* An accent border */
}

h1, h2 {
    color: #0056b3;
}
