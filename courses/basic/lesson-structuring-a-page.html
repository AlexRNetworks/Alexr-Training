Lesson 2: Structuring a Web Page

Every HTML document has a standard "boilerplate" structure that acts as its foundation. Think of this as the minimum required code for any webpage.

The HTML Boilerplate

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <!-- Your content goes here! -->
</body>
</html>

Key Structural Tags Explained

Inside the <body>, we use "semantic tags" like <header>, <main>, and <footer> to give meaning to our layout. This helps with accessibility and SEO.
