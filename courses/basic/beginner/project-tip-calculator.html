Project: Interactive Tip Calculator

It's time to build your first interactive web application! You will use your knowledge of HTML, CSS, and JavaScript to create a functional tip calculator.

Project Requirements

  • Input fields for the bill amount and tip percentage.
  • A "Calculate" button.
  • An area to display the calculated tip amount and the total bill.
  • Your JavaScript code should get the values from the inputs, perform the calculation, and update the display area when the button is clicked.

Project Setup

We recommend using the live editor from the Basic Course final project to build this. Click the button below to go directly to the editor.

Go to Project Editor

HTML Starter Code:

You can use this HTML as a starting point in the editor.

<!-- You can use this HTML as a starting point -->

<label for="billAmount">Bill Amount:</label>
<input type="number" id="billAmount">
<br><br>

<input type="number" id="tipPercentage">
<br><br>
<button id="calculateBtn">Calculate</button>
<h3>Results</h3>
<p>Tip: <span id="tipAmount">...</span></p>
<p>Total: <span id="totalBill">...</span></p>
