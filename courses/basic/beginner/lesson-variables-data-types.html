Lesson 2: Variables & Data Types

To do anything useful in programming, we need a way to store and label information. We do this with variables. Think of a variable as a labeled box where you can keep a piece of information to use later.

Declaring Variables

In modern JavaScript, we use the let and const keywords to declare variables. Use let for variables whose value might change, and const for variables that will always stay the same (constants).

let userScore = 0; // We can change this later
userScore = 10;      // This is okay

const siteName = "Alexr Training"; // This can't be changed
// siteName = "Something Else"; // This would cause an error!

Basic Data Types

Variables can hold different types of data. The most common ones are:

  • String: A sequence of text, wrapped in quotes. e.g., "Hello, world!"
  • Number: Any kind of number, including integers and decimals. e.g., 42 or 3.14
  • Boolean: A simple true or false value. e.g., true or false
let greeting = "Welcome!";      // This is a String
const lessonsInCourse = 5;      // This is a Number
let isLoggedIn = false;         // This is a Boolean

Understanding these basic building blocks is essential before we can start making our pages interactive.
