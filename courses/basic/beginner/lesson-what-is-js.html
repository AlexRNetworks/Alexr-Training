Lesson 1: What is JavaScript?

If HTML is the skeleton and CSS is the skin, then JavaScript (JS) is the brain and nervous system. It is a full programming language that allows you to make your websites interactive and dynamic.

With JavaScript, you can react to user actions like clicks and keyboard presses, fetch data from other websites, create animations, and much more. It's what transforms a static document into a web application.

You can include JavaScript in your HTML file by using the <script> tag. It's best practice to place it at the very end of your <body> tag to ensure all HTML content has loaded before the script runs.

<body>
    <!-- All your HTML content -->
    
    <script src="/assets/js/my-script.js"></script>
</body>
