Lesson 4: Introduction to the DOM

This is where JavaScript becomes truly powerful for web development. The Document Object Model (DOM) is a programming interface for web documents. In simple terms, it's a representation of your HTML page that JavaScript can read and manipulate.

What is the DOM?

When a browser loads a webpage, it creates a "model" of that page in memory. It sees your HTML as a tree of objects (or "nodes"). The DOM allows our JavaScript to interact with this tree. We can find elements, change their content, modify their styles, and even create or delete them entirely!

Selecting and Modifying Elements

The most common task is selecting an element and doing something with it. We can use document.querySelector() to find the first element that matches a CSS selector.

Imagine your HTML has a heading: <h1 id="main-title">Hello World</h1>

// 1. Select the element using its ID
const myHeading = document.querySelector("#main-title");

// 2. Change its text content
myHeading.textContent = "Welcome to Alexr Training!";

// 3. Change its style
myHeading.style.color = "blue";

After this script runs, the heading on your page will instantly change its text and color. This ability to manipulate the DOM is the foundation of every interactive website, from simple alert boxes to complex web applications.
