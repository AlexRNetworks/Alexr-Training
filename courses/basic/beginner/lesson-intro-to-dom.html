Lesson 4: Introduction to the DOM

The Document Object Model (DOM) is a representation of your HTML page that JavaScript can read and change. This is how we make pages interactive.

Selecting and Modifying Elements

We use document.querySelector() to find an element. Once we have it, we can change its properties, like its text or style.

Imagine your HTML has: <h1 id="main-title">Hello World</h1>

// 1. Select the element
const myHeading = document.querySelector("#main-title");

// 2. Change its text content
myHeading.textContent = "Welcome to Alexr Training!";

// 3. Change its style
myHeading.style.color = "blue";
