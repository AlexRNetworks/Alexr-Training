Lesson 3: Functions & Control Flow

We need ways to perform actions and make decisions. We do this with functions and control flow statements like if...else.

Functions: Reusable Blocks of Code

A function is a block of code designed to perform a particular task. You can "call" a function whenever you need it to run.

// Defining the function
function greetUser() {
  console.log("Hello, there!");
}

// Calling the function
greetUser();
