Lesson 3: Functions & Control Flow

Now that we can store data, we need ways to perform actions and make decisions. We do this with functions and control flow statements.

Functions: Reusable Blocks of Code

A function is a block of code designed to perform a particular task. You can "call" a function whenever you need it to run, which prevents you from having to write the same code over and over again.

// Defining the function
function greetUser() {
  let userName = "Alex";
  console.log("Hello, " + userName + "!");
}

// Calling the function to make it run
greetUser(); // This will print "Hello, Alex!" to the console.

Control Flow: Making Decisions with `if...else`

Often, you only want to run code if a certain condition is true. The if...else statement is perfect for this. It allows your code to respond differently to different situations.

let userIsLoggedIn = true;

if (userIsLoggedIn === true) {
  console.log("Welcome back!");
} else {
  console.log("Please log in to continue.");
}

In this example, because userIsLoggedIn is true, only the first message will be printed. If it were false, the second message would be printed instead. This is the foundation of creating responsive, intelligent web applications.
