Lesson 4: The CSS Box Model

Every single element on a web page is a rectangular box. Understanding how this box is structured is the most important key to mastering CSS layouts. This structure is called the Box Model.

Margin
Border
Padding
Content

The Four Parts Explained

  • Content: The actual content of the box, where your text or images appear.
  • Padding: The transparent space between the content and the border.
  • Border: A line that goes around the padding and content.
  • Margin: The transparent space outside the border, pushing other elements away.
