Lesson 1: Introduction to HTML

Welcome to your first real lesson! Let's start at the very beginning. HTML, which stands for HyperText Markup Language, is the standard language used to create and design documents on the World Wide Web. It's the skeleton of every website.

HTML isn't a programming language; it's a markup language. This means it uses tags to "mark up" or define the structure of content. You tell the browser what something is (like a heading, a paragraph, or an image), and the browser displays it accordingly.

HTML Elements and Tags

HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller, and so on.

A tag is simply a keyword wrapped in angle brackets, like <p>. Most tags come in pairs: an opening tag and a closing tag. For example, to mark up a paragraph, you would use an opening <p> and a closing </p>. 

<p>This is a paragraph of text.</p>
<h1>This is a main heading.</h1>

In the example above, you can see the two most common tags: the paragraph tag <p> and the heading tag <h1>. The number in the heading tag can range from 1 to 6, with <h1> being the most important and largest. Try it out!
