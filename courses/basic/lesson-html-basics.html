Lesson 1: Introduction to HTML

Welcome to your first real lesson! HTML, which stands for HyperText Markup Language, is the standard language used to create and design documents on the World Wide Web. It's the skeleton of every website.

HTML Elements and Tags

HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content. A tag is simply a keyword wrapped in angle brackets, like <p>. Most tags come in pairs: an opening tag and a closing tag.

<p>This is a paragraph of text.</p>
<h1>This is a main heading.</h1>
