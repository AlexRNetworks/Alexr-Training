Lesson 3: Introduction to CSS

So far, our pages have structure, but they lack style. That's where Cascading Style Sheets (CSS) comes in. If HTML is the skeleton of a webpage, CSS is the skin, clothes, and overall appearance. It handles all the colors, fonts, layouts, and visual design.

What is a CSS Rule?

CSS works by defining "rules". A rule selects an HTML element and tells the browser how to style it. The basic syntax looks like this:

selector {
    property: value;
}

Three Ways to Add CSS

There are three ways to include these CSS rules in your project, but one is highly preferred.

1. External CSS (Best Practice)

You create a separate file with a .css extension (like our own style.css file!) and link to it from your HTML file using a <link> tag inside the <head>. This is the best method because it separates content (HTML) from presentation (CSS) and allows you to style many pages with a single file.
