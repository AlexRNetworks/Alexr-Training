Lesson 3: Introduction to CSS

If HTML is the skeleton of a webpage, Cascading Style Sheets (CSS) is the appearance. It handles all the colors, fonts, layouts, and visual design.

What is a CSS Rule?

CSS works by defining "rules" that select an HTML element and tell the browser how to style it. The basic syntax is selector { property: value; }.

p {
    color: blue;
    font-size: 16px;
}

Three Ways to Add CSS

You can add CSS using an external stylesheet (best practice), an internal style block in the head, or inline styles on an element. We use an external stylesheet for this site!
