Lesson 3: Introduction to React & JSX

React is a JavaScript library for building user interfaces. It was created by Facebook and is now the most popular UI library in the world. Its main idea is to allow developers to build complex interfaces by breaking them down into small, reusable pieces called components.

Why Use React?

  • Component-Based: Build encapsulated components that manage their own state, then compose them to make complex UIs.
  • Declarative: You tell React what you want the UI to look like, and it handles the complex DOM manipulation to make it happen.
  • Learn Once, Write Anywhere: The principles of React can be used to build web apps, mobile apps (with React Native), and even desktop apps.

What is JSX?

React introduces a syntax extension for JavaScript called JSX (JavaScript XML). It lets you write HTML-like code directly in your JavaScript files. This makes creating component templates much more intuitive.

// This is JSX - it looks like HTML, but it's really JavaScript!
const myElement = <h1>Hello, React!</h1>;

// In a real React app, you would render this element to the DOM.
// For example:
// const root = ReactDOM.createRoot(document.getElementById('root'));
// root.render(myElement);

While you can use React without JSX, it's a core part of the experience and what we'll be using moving forward. In the next lesson, we'll see how to use JSX to build our first React components.
