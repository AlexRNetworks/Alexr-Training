Project: User Profile Card App

It's time to combine everything you've learned in the intermediate course: async JavaScript, APIs, and the basics of React. Your task is to build a simple application that fetches a user's data from a public API and displays it as a profile card.

Project Requirements

  • Use the `fetch` API to get user data from `https://reqres.in/api/users/2`.
  • Create a React component called `UserProfileCard`.
  • Use the `useState` hook to store the fetched user data.
  • Use the `useEffect` hook to fetch the data when the component first loads.
  • Display the user's avatar (image), first name, last name, and email in your component.
  • Show a "Loading..." message while the data is being fetched.

Project Setup

This project requires a slightly different editor setup. We recommend clearing the HTML and CSS tabs in the editor and working only in the "JavaScript" tab (which you can create in your own editor). You will also need to add the React library links to the HTML.

Go to Project Editor

HTML Starter Code:

In the HTML tab of the editor, use this simple setup:

<!-- Add this to the 'Settings' or as external scripts -->

<script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>

<!-- Your React app will render in this div -->

<!-- Your React code will go in the JS tab, with type="text/babel" -->
